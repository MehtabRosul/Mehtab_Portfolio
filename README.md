# Portfolio V2 - Mehtab Aftabur Rosul

A world-class, futuristic portfolio website showcasing AI/ML expertise, LLM engineering, full-stack development, and quantum computing passion.

## Tech Stack

- **Framework**: Next.js 14+ (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion, GSAP, Three.js
- **Theme**: next-themes (Light/Dark/System)
- **3D**: React Three Fiber

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```
